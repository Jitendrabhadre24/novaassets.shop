# **App Name**: NovaAssets

## Core Features:

- Product Catalog Display: Showcase digital products with images, titles, and descriptions in a grid layout on the homepage and dedicated listing pages.
- Product Detail View: A dedicated page for each product to display full descriptions, features lists, and high-resolution images.
- Category Navigation: Enable users to browse and filter products by predefined categories such as Ebooks, Courses, Templates, and more.
- External Unlock & Download Flow: Manage the user journey for unlocking product access via an external redirect, followed by a modal for direct download upon return.
- Responsive & Premium UI: Implement a fully responsive design for seamless viewing on mobile and desktop, adhering to a clean SaaS aesthetic with premium spacing, shadows, and hover effects.
- Static Product Data: Utilize hardcoded dummy product data, including image URLs, to populate the marketplace for demonstration and MVP purposes.
- AI-powered Product Recommendation Tool: A tool that offers personalized digital asset recommendations to users based on simulated browsing patterns or categorized interests on the homepage.

## Style Guidelines:

- Primary interactive color: A vibrant digital purple (#8C40FF) for calls-to-action and key highlight elements, evoking a premium and modern feel against the dark theme.
- Background color: A very dark, near-black hue (#171519) with a subtle purple undertone to provide depth and align with the requested 'dark premium theme (black background)'.
- Accent color: A cool, bright blue-violet (#A7A1FF) to complement the primary purple, used for secondary emphasis and interactive elements, enhancing the digital aesthetic.
- Body and headline font: 'Inter' (sans-serif) for a modern, objective look, with bold variations for headings. Note: currently only Google Fonts are supported.
- Sleek, minimalistic outline icons that maintain a clean and contemporary SaaS premium aesthetic.
- Consistent premium UI spacing, utilizing a flexible grid system that adapts from 3 columns on desktop to 1 column on mobile devices.
- Smooth hover effects, subtle loading skeletons, and understated card shadows and glow effects to enhance the premium user experience.
